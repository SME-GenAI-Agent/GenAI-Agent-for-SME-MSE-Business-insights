# GenAI-Agent-for-SME-MSE-Business-insights
This repository hosts an AI agent designed to empower SME/MSME owners with data-driven decision-making. It uses LangChain and Llama-3 to answer questions about sales, generate profit/loss summaries, and suggest strategies for inventory optimization and cost savings, acting as a virtual business consultant
